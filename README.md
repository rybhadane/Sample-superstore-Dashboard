# Sample-superstore-Dashboard 
## ExcelR Assignment
This dataset is usually used for getting handson experience on various dashboards

## KPI cards on Tableau Dashboard
1. Sales
2. Profit Margin
3. Return Rate

## Charts on Tableau Dashboard
1. Bar chart (Sales by Category and Sales by Sub-Category)
2. Line chart (Sales trend)
3. Pie Chart (Sales by Segment)
4. Map Chart (Sales by States)
5. Table (Top 5 Products)

## Interaction 
1. Metric (Sales, Profit, # of Orders)
2. Filter by Region (Central, East, South and West)
3. Filter by Year of Order Date
